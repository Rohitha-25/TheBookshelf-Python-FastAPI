# 📚 TheBookshelf – Book Management API (FastAPI)

## Overview
TheBookshelf is a backend REST API built using FastAPI for managing books, authors, and user interactions. It demonstrates scalable backend architecture with secure API design and efficient data handling.

## Tech Stack
- Python
- FastAPI
- SQLAlchemy (ORM)
- MySQL
- Pydantic
- Uvicorn

## Features
- CRUD operations for books and authors
- Data validation using Pydantic
- High-performance API using FastAPI
- Automatic API documentation (Swagger UI)
- Modular and scalable project structure

## Key Concepts Implemented
- RESTful API design
- Schema validation
- Dependency injection
- Asynchronous request handling
